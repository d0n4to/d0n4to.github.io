Auto Minder

Questo codice Java crea un'interfaccia grafica per l'inserimento e la verifica di dati relativi a veicoli. Consente agli utenti di inserire informazioni come nome, targa, e altri dettagli, e utilizza un thread per validare i dati inseriti, mostrando messaggi di errore in caso di problemi. L'interfaccia è personalizzata con font, colori e layout specifici.
